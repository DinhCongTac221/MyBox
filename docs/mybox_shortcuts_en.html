When focus is in "Text Input" control, Delete/Home/End/PageUp/PageDown/Enter/Ctrl-c/v/z/y/x work for texts in the focus. Or else shortcuts work for interface.
When focus is not in "Text Input" control, Ctrl/Alt can be omitted. Example, if an image is currently foused, press "c" to copy and press "2" to set as pane size.
